MAP MENU

The "Systemapic" button serves as a progress feedback, and as a Map Menu button.

When the button has been clicked, a menu with different features will appear. The content of this menu will depend on the access level of the user. This is also where future plug-ins will be installed, such as the media library.

MAP MENU

PROJECTS

Under the "Projects" pane you will find clients and projects alike.

If you have write access to a project, you can change the title, description and featured image by double clicking on these containers. All changes are automatically saved to the database.

If you wish to create a new project or client, you can do so by clicking on the [+] buttons. The [+] in the client drop-down creates a new project for that client, and the [+] at the bottom of the pane creates a new client.

An url is then automatically created for the client/project, at systemapic.com/clientname/projectname.

When you hover your mouse pointer over a client/project, you will see that a box with an "i" appears in the top right corner. If you put your mouse over this symbol you will see some general project information.

If you have manager access to project, you will also be able to delete the project (don't worry, you will be asked to confirm it twice before it's actually gone).

To delete a client, you will have to delete all projects connected to that client first.

MAP MENU - OPTIONS

CONNECTED ACCOUNTS

Here you can connect one or several MapBox accounts.

By default every project will be connected to the Systemapic MapBox account that contains a few base layers.

MAP MENU - OPTIONS

CONTROLS

Here you can enable/disable the tool boxes that appears on the actual map.

Zoom
enables zooming on the map, and puts a [+] and a [-] button in the top left corner of the map.

Draw
enables a draw toolbox, where users can draw simple shapes on the map.

Inspect Layers
enables the box in the bottom right corner, where users can change the order, isolate, and disable activated layers.

Layer Menu
enables the layer selector.

Legends
enables layer legends.

Scale
the scale tool in the top right corner.

Geolocation:
search for address or location

Baselayer Toggle
toggle between satellite, terrain, or which ever layers that's been set as base layers Mouse Position: Tells the long/lat coordinates of the mouse pointer.

MAP MENU - OPTIONS

POSITION

Sets the landing position of the map.

Zoom in to a position, and click the "Set current position" button, or plot in the coordinates manually.

MAP MENU - OPTIONS

BOUNDS

Locks bounds and zoom of the map.

When bounds are set, the user will not be able to move outside of these coordinates on the map.

Here you can also force max/min zoom levels.

MAP MENU - OPTIONS

LAYER MENU

Here you will find all acceptable shapefiles/raster files that's been uploaded to the project through the "Data Library" under the "Data library" section.

When the "Layer Menu" has been activated, and you have selected "Layer Menu" under the "Controls" option, you will be able to add these layers to the map by clicking on them.

When you have the "Layer Menu" pane activated, a button called "ADD FOLDER" will appear on the top of the Layer Menu.



Illegeal layer structure:

The layer folders enables you to group layers, and hide/open these groups of layers.

To "put" a layer in a folder, hover your mouse over the layer, and click the [<] or [>] button. Any layer or layer folder that's indented more than its parent folder will be "foldable" by the layer folder.

However, it's important to keep the hierarchical structure correct! The example on the bottom right of this page is an illegal layer structure. Make sure that indented layers have a corresponding layer folder.

To change the order of the layers, you can simply dragging-and-drop them.

If you want to change the name of a layer or a layer folder, double click on it.

MAP MENU - OPTIONS

BASE LAYERS

These are the default base layers that you wish to use for the project.

You may use base layers from a MapBox account, or you can use layer files that you have uploaded yourself.

You may have as many base layers as you wish.

Layers selected for base layers will not be available to use in the Layer Menu section, and vice versa.

If you have selected the "Base Layer Toggle" option in the "Controls" pane, users can turn the different layers on and off. They may turn off all base layers, to look at a layer isolated if they wish to.

LEGENDS

Legends attached to layer will appear here.Layer legends gets activated through "Map Menu > Options > Controls > Legends"

LAYER INSPECTOR

Layer Inspector activated through "Map Menu > Options > Controls > Layer Inspector".

All activated layers (from user side) will appear here. The arrows to the left of the layer name will change layer order (z-index). The little eye to the right will isolate layer (hide other layers), and the [X] will disable layer, and remove it from the list.

MISCELLANEOUS

Project Header

Unique header for each project. Double click on title, or project description to change content. Click on image to upload.

This is not the "official" project name, so it's not reflected in the url. To do this you have to set project name in "Map Menu > Projects".

Layer Info

Info box for each layer. Gets activated through "Map Menu > Options > Controls > Description". Double click in content area to add text. See section "Docs" for text editing details.

Mouse position

Tells the geo location of the mouse pointer. Gets activated through "Map Menu > Options > Controls > Mouse Position"

Scale

Tells the scale of the map. Gets activated through "Map Menu > Options > Controls > Scale"

DOCUMENTS

Double click on the "Title" to change it's name. Double click on the "Text content" to start editing the text.

Hit enter twice to get a horizontal separator line.

When you start typing, select a word or a phrase, and you will see a tool-tip with styling options appear.

DATA LIBRARY

The datalibray is a fully automated "intelligent" file handling system, that automatically organized your files into the database.

Shapefiles will automatically be available as layers or base layers, and can be accessed from "Options > Layer Menu" and "Base Layers"

Image files will be available in the image library shortly.

Double click on file names if you wish to rename them

Uploading files

There are two ways to upload files: to click the "upload" button, or to drag-and-drop the files you wish to upload onto the screen.

If you zip a folder with several files, these files will come as one entry, containing several files. This is particularly handy when it comes to shapefiles, where several files depend on each other. Because of this, shapfiles HAVE TO BE ZIPPED before you upload them!

When uploading large files, you will see a slim white progress bar at the top of the screen. Do not reload the page, or navigate to another page when you are uploading files.

Downloading files

All files are available for download. Select the files you wish to download, and click the "Download" button. You will then be taken to a screen that shows you some basic information about the files you are about to download. When you click "Download" again, the files will be zipped into an archive, and made available for downloading. When the files are ready, you will be taken to a final screen with yet another "Download" button.

USER MANAGEMENT

This is where you create users, and delegate access to the different projects.

When you create a user, you first need to fill in the basic information, containing First Name, Last Name, and email address. A password will then be generated and emailed to the user. (Users cannot choose their own passwords for security reasons).

Once a user is created you can continue to fill inn other information such as company name, phone number, and position.

In the last field you will notice it says "0 project(s)". If you wish to give a user access to one or several projects, click on this field, and select what projects you wish to grant access to.

Read
gives read only access to user. Can upload files, but not delete them. Cannot see other users.

Edit
gives access to edit project, including editing layers, upload and delete, but cannot create users.

Manager
gives access to edit project, and create users. Cannot see admins.

Super Admins
gets full access to all projects, and can see all users on all projects. Super Admins cannot be created in the user interface, and have to be manually hard coded on the server.
